# Submitting Issues

To submit an issue (Bug, Improvement, Request) please select one of the options below.
Please leave the [TYPE] angled bracket text in the title to make the Issues easier to categorize

## Error Report
Use this template to report errors in the source code, the book or the repository  
[Create an Error Report](https://github.com/BPP-Thesis-Artifact/Files-Public/issues/new?template=report_error.yml)

## Suggest Improvement
Use this template to suggest improvements to explanations, examples, structure...  
[Suggest an Improvement](https://github.com/BPP-Thesis-Artifact/Files-Public/issues/new?template=suggest_improvement.yml)

## Request Content
Use this template to request new topics, examples, etc. you would like to see in the future  
[Request new Content](https://github.com/BPP-Thesis-Artifact/Files-Public/issues/new?template=request_content.yml)

## Other Issue
Use this template to report an Issue that is not covered under the provided templates  
[Other Issue](https://github.com/BPP-Thesis-Artifact/Files-Public/issues/new?template=other_issue.yml

Check your issue periodically to see if its status has changed. If your issue is marked with “Status: needs-info,” please add additional information by editing the text of the issue rather than providing the information in a comment.

# Note
This repository does not support Pull-Request based submissions at this time
